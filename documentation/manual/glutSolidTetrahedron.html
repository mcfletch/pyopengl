glutSolidTetrahedron

render a solid or wireframe tetrahedron (4-sided regular solid) respectively.

Signature

glutSolidTetrahedron( )-> void
glutSolidTetrahedron( ) 
glutSolidTetrahedron(  ) -> None
glutWireTetrahedron( )-> void
glutWireTetrahedron( ) 
glutWireTetrahedron(  ) -> None

Description

glutSolidTetrahedron and glutWireTetrahedron render a solid or wireframe tetrahedron respectively centered at the modeling coordinates origin with a radius of 3 .
