glCreateFramebuffers

create framebuffer objects

Signature

glCreateFramebuffers( GLsizei ( n ) , GLuint *( framebuffers ) )-> void
glCreateFramebuffers( n , framebuffers )

Parameters

VariablesDescription
n
Number of framebuffer objects to create.
framebuffers
Specifies an array in which names of the new framebuffer objects are stored.

Description

glCreateFramebuffers returns n previously unused framebuffer names in framebuffers , each representing a new framebuffer object initialized to the default state.

Errors

GL_INVALID_VALUE is generated if n is negative.

Version Support

glCreateFramebuffers
